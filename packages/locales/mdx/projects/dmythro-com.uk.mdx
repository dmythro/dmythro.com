Цей вебсайт — персональне портфоліо, побудоване на **[Astro](https://astro.build/)**, **[DaisyUI](https://daisyui.com/)** та **[Tailwind CSS](https://tailwindcss.com/)**. Він слідує філософії **«нуль JavaScript за замовчуванням»** — сторінки рендеряться як статичний HTML з CSS-only інтерактивними компонентами, а **React**-острови використовуються лише там, де дійсно потрібна клієнтська інтерактивність.

Сайт **двомовний** (англійська та українська) з повною підтримкою **i18n**, включаючи визначення мови, перекладений контент та локалізовані маршрути. **Перемикач теми** циклічно змінює три стани: системна, темна та світла. Дизайн повністю адаптивний та оптимізований для друку, зі спеціальним CSS, що створює чисті сторінки CV для **експорту в PDF**.

Проект організований як **монорепо** на основі **[Bun](https://bun.sh/)** та **[Turborepo](https://turbo.build/repo)**. Спільні пакети надають константи, переклади та markdown-контент, які використовуються **Astro**-застосунком. Якість коду забезпечується **[Biome](https://biomejs.dev/)** для форматування та лінтингу, з суворим **TypeScript** повсюди.

Сайт розгорнуто на **[Cloudflare Pages](https://pages.cloudflare.com/)** з автоматичними збірками з **GitHub**. Продуктивність — пріоритет: оцінки **Lighthouse** стабільно високі завдяки **статичній генерації**, мінімальному JavaScript та оптимізованим ресурсам.
